What's new

What changes, release after release

Accord updates itself: a banner tells you inside the app, one click installs the new version. Here is what each recent release brings you.

3.2.0 July 19, 2026 GitHub release →
  • A more useful server menu — "Mark as read" now sits at the top of the menu whenever unread channels remain, and a new "Edit my server profile" entry jumps straight to your per-server nickname and avatar.
  • Your status in the bottom panel — the bottom-left panel now shows your presence (Online, Idle, Do not disturb…) or your custom status instead of your friend code. The friend code still lives in Settings → My account and in the Friends tab.
3.1.0 July 19, 2026 GitHub release →
  • Redesigned server menu — hovering or keyboard-navigating fills each row with blurple (red for "Leave server"), with white text and icons: easier to read, closer to what you already know.
3.0.0 July 19, 2026 GitHub release →

A big release focused on reliability and writing comfort — fully compatible with friends still on 2.x.

More reliable connections

  • Instant reunions — Accord remembers each friend's last known address and dials it at startup or when your Mac wakes up: sessions come back without waiting for network discovery.
  • Friend banners and avatars, finally dependable — the reconnection race that could swallow the profile announcement was fixed at the root.
  • Per-friend diagnostics — the Network panel shows, friend by friend, whether a session is active and the last address used.

Writing and reading

  • Markdown tables and task lists, a Copy button on code blocks, the full date when hovering a message time.
  • Copy a whole conversation as clean Markdown, grouped by day.
  • "Back to latest" — a floating button brings you back to the newest message after a long scroll.
  • Voice message playback speed: 1×, 1.5× or 2×.

Personalization

  • Custom theme editor — pick your own colors on a dark or light base, the rest of the palette is derived automatically; share the result with a simple accord-theme:… code.
  • Interface font choice (System, Rounded, Serif) and scheduled Do Not Disturb.
  • Smooth scrolling through long conversations, even with thousands of loaded messages.

The full version-by-version history lives in the project changelog.
